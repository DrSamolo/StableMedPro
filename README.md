# StableMed CRM

Ce repo conserve la structure d'origine avec le CRM dans `StableMed/`.

## Lancer en local (recommandé)

```bash
npm install
npm run dev
```

Ces commandes délèguent automatiquement vers `StableMed/`.

## Structure

- `StableMed/` : application CRM
- `current_before_restore/` : sauvegarde de l’état précédent (sécurité)
